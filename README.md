----------------------------------------------------------------------------
Asteroid Shooting Game
----------------------------------------------------------------------------

You should install raylib to your system for execute the project


----------------------------------------------------------------------------
For Linux;

Fedora 

    sudo dnf install raylib raylib-devel
Arch

    sudo pacman -S raylib
Ubuntu

    sudo apt install libraylib-dev
----------------------------------------------------------------------------
For macOS;

    brew install raylib
----------------------------------------------------------------------------
For Windows(MSYS2 Terminal);

    pacman -S mingw-w64-x86_64-raylib
----------------------------------------------------------------------------

After That installation process you can just open the project root folder and execute this command

    make
